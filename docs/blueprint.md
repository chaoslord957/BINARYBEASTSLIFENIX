# **App Name**: LifeNix

## Core Features:

- AI Symptom Checker: An AI-powered chatbot that analyze user-reported symptoms and suggests possible conditions, recommends doctor specialties, and provides disclaimers to consult a real doctor. Allow the previous file that were uploaded to be put on them.
- Medical Report Summarisation: Tool that extracts text from uploaded medical reports (images/PDFs) using OCR and summarises the content using AI, highlighting any abnormal values. Give Diet plan for the people if they have any problem after you OCR your report.
- Live Health Monitor: Fetches and displays real-time health data (heart rate, steps, SpO2) from user's wearable devices via Google Fit and Apple HealthKit.
- Doctor Consultation Booking: Allows users to search for doctors by name, specialty, or symptom, filter by online availability and category, and book appointments.
- Medicine Delivery: Enables users to search for medicines, view product details, add items to cart, and complete the checkout process for medicine delivery.
- Medical Vault: Provides a secure storage for medical documents (reports, prescriptions, insurance cards) with file upload, viewer, and AI-extracted insights.
- Ambulance Service Booking: Lets users book an ambulance with map interface showing current location and destination, along with real-time tracking.
- GymFreak and Nutritions: A Nutritional diet plan that will be given by the AI when a user prompts something or wishes for a DIET plan, A Balanced Diet PLAN
- Articles and Daily Blogs: A Daily BLOG posts updated by people, that would be shown to people, kind of health news that would be given.

## Style Guidelines:

- Primary colour: Soft blue (#A0D2EB) to evoke a sense of calm and trustworthiness, inspired by the need for a 'peaceful' UI.
- Background colour: Light grey (#F0F4F8), nearly white, for a clean and spacious feel, consistent with the desired aesthetic of 'plenty of whitespace'.
- Accent colour: Light green (#BCE29E), analogous to the primary, suggesting growth and health, and providing visual contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif that offers a balance of modernity and warmth, ensuring readability and a friendly feel.
- Use a set of minimalist, consistent icons, with thin lines and rounded corners, to convey information clearly and complement the clean design.
- Implement a card-based layout with generous padding and whitespace to organise information and maintain a smooth, uncluttered user experience, in keeping with the design goals mentioned in the prompt.
- Subtle animations and transitions to provide visual feedback and enhance user engagement without disrupting the peaceful and smooth aesthetic.
- Keep Everything in Indian Style, the names the currency.
- The photo should be kept in Indian Style.
